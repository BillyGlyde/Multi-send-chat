----------------

Purpose
-------
MultiSend Chat is a Tkinter-based desktop automation tool that:
- Detects windows using wmctrl
- Activates windows using xdotool
- Sends typed messages to multiple windows
- Supports ChatGPT, Claude, Perplexity, terminals, and text editors
- Uses clipboard probing for Perplexity input focus
- Allows global cancel using Escape via pynput
- Scrollable window list via Tkinter canvas
